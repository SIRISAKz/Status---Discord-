# Status---Discord
โชว์สเตตัสบนดิสคอร์ด สถานะของคุณแบบเท่ห์ๆ 

# Setup & Use  
 1). กรอก token discord ของเราลงไปในไฟล์ setting.json 
 
 2). กรอกข้อมูลต่างๆ ที่จะให้โชว์บนสถานะ เช่น name log url web
 
 3). รันโปรแกรมเพื่อให้โชว์สถานะ **( py main.py )**
